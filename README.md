# Midnight Prism

A dark, cosmic-toned theme for Zed — ported from the [Andromeda Colorizer](https://github.com/EliverLara/Andromeda) VS Code theme.

## Color Palette

| Scope | Color |
|---|---|
| Background | `#23262E` |
| Foreground | `#D5CED9` |
| Keywords | `#c74ded` Purple |
| Functions / Types | `#FFE66D` Yellow |
| Strings | `#96E072` Green |
| Variables / Properties | `#00e8c6` Cyan |
| Numbers / Attributes | `#f39c12` Orange |
| HTML Tags | `#f92672` Hot Pink |
| React Components | `#f65e49` Coral Red |
| Constants / Operators | `#ee5d43` Red |
| Regex | `#7cb7ff` Blue |
| Comments | `#A0A1A7` Gray |

## Installation

Available on the [Zed Extension Store](https://zed.dev/extensions?query=midnight-prism).

Or manually: copy `themes/midnight-prism.json` to `~/.config/zed/themes/`.
