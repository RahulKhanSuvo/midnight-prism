## Installation

### Via Zed Extension Store *(recommended)*

1. Open Zed and press `Ctrl+Shift+X` to open Extensions
2. Search for **Midnight Prism**
3. Click **Install**
4. Go to `Zed ‚Üí Settings ‚Üí Theme` and select **Midnight Prism**

### Manual Install

1. Copy `themes/midnight-prism.json` to your Zed themes folder:

```sh
cp themes/midnight-prism.json ~/.config/zed/themes/
```

2. Open `~/.config/zed/settings.json` and set:

```json
{
  "theme": {
    "mode": "dark",
    "dark": "Midnight Prism"
  }
}
```

## Features

- **Cyberpunk-inspired neon palette** ‚ÄĒ vivid, saturated accent colors on a deep dark background
- **Comfortable dark backgrounds** ‚ÄĒ easy on the eyes during long coding sessions
- **Full semantic color coverage** ‚ÄĒ error, warning, success, git status, and conflict states all properly themed
- **Neon-pink active line number** ‚ÄĒ instantly know which line your cursor is on
- **Cyberpunk terminal colors** ‚ÄĒ matching neon ANSI palette for the integrated terminal
- **8 player cursors** ‚ÄĒ distinct neon colors for collaborative editing
